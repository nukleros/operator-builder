# Operator Builder

**Accelerate the development of Kubernetes operators.**

Operator Builder is used to rapidly build operators that manage application
stacks.

See our [documentation website](https://opeator-builder.io/) for more
information.

## Developer Docs

If you're interested in contributing to Operator Builder, please see our
[Contributing
Guide](https://github.com/nukleros/operator-builder/blob/main/CONTRIBUTING.md)
and the [developer
documentation](https://github.com/nukleros/operator-builder/tree/main/docs/dev).
